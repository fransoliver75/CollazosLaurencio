HTML 2.0

1995 - PRIMERA ESTANDARIZACIÓN

Características Principales

  • Primera especificación formal de HTML por el IETF
  • Soporte para formularios básicos con elementos como input y textarea
  • Inclusión de tablas simples para organización de contenido
  • Elementos de texto fundamentales: p, h1-h6, strong, em
  • Listas ordenadas (ol) y no ordenadas (ul) para presentar información
  • Enlaces e imágenes con los elementos a e img para navegación y multimedia
  • Estructura básica de documento con head y body
Ejemplo de código HTML 2.0
<!DOCTYPE HTML PUBLIC "-//IETF//DTD HTML 2.0//EN">
<html>
<head>
  <title>Mi primera página HTML 2.0</title>
</head>
<body>
  <h1>Bienvenido a mi sitio web</h1>
  <p>Esta es una página HTML 2.0 que muestra las características básicas.</p>
  <a href="otra_pagina.html">Enlace de ejemplo</a>
  <img src="imagen.gif" alt="Descripción">
</body>
</html>

HTML 3.2

1997 - ESTÁNDAR W3C

Características Principales

  • Primera recomendación de HTML por el W3C (World Wide Web Consortium)
  • Tablas más complejas con atributos como colspan y rowspan para diseño avanzado
  • Soporte para applets de Java para interactividad básica
  • Mejoras en el flujo de texto alrededor de imágenes con el atributo align
  • Atributos para alineación y dimensiones de elementos para mejor control visual
  • Soporte inicial para scripts con el elemento script
  • Elementos de fuente (font) para control tipográfico básico
  • Mejoras en formularios con nuevos tipos de controles
Ejemplo de código HTML 3.2
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 3.2 Final//EN">
<html>
<head>
  <title>Página HTML 3.2 con tabla</title>
</head>
<body>
  <table border="1" cellpadding="5">
    <tr>
      <td colspan="2">Título de tabla</td>
    </tr>
    <tr>
      <td>Celda 1</td>
      <td>Celda 2</td>
    </tr>
  </table>
  <applet code="MiApplet.class" width="300" height="200">
    Tu navegador no soporta applets de Java.
  </applet>
</body>
</html>

HTML 4.01

1999 - SEPARACIÓN DE PRESENTACIÓN

Características Principales

  • Introducción de hojas de estilo en cascada (CSS) para separar contenido y presentación
  • Marcado mejorado para accesibilidad con atributos como alt, title y accesskey
  • Mejoras significativas en formularios con nuevos controles y validación
  • Separación clara entre estructura y presentación para un desarrollo más limpio
  • Soporte para frames (iframe) para contenido embebido
  • Scripting mejorado con soporte para DOM (Document Object Model)
  • Tres variantes: Strict (sin elementos de presentación), Transitional (con elementos obsoletos) y Frameset (para frames)
  • Mejor internacionalización con soporte para diferentes conjuntos de caracteres
Ejemplo de código HTML 4.01
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
<html>
<head>
  <title>Página HTML 4.01 con CSS</title>
  <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
  <link rel="stylesheet" type="text/css" href="estilos.css">
</head>
<body>
  <div id="contenedor">
    <h1>Título principal</h1>
    <p class="destacado">Este párrafo tiene una clase CSS aplicada.</p>
    <iframe src="marco.html" width="300" height="200"></iframe>
  </div>
</body>
</html>

XHTML 1.0

2000 - HTML COMO XML

Características Principales

  • Reformulación de HTML 4 como aplicación XML para mayor consistencia
  • Sintaxis más estricta y bien formada siguiendo las reglas de XML
  • Elementos deben estar correctamente anidados y cerrados explícitamente
  • Atributos deben ir entre comillas (simples o dobles) obligatoriamente
  • Elementos vacíos deben cerrarse con /> (ej: <br />)
  • Nombres de elementos y atributos en minúsculas obligatoriamente
  • Mejor compatibilidad con otros lenguajes XML y herramientas de procesamiento
  • Documentos bien formados que pueden ser analizados por parsers XML
Ejemplo de código XHTML 1.0
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="es" lang="es">
<head>
  <title>Página XHTML 1.0 Estricta</title>
  <meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
</head>
<body>
  <h1>Ejemplo XHTML</h1>
  <p>Este es un párrafo correctamente formado.</p>
  <br />
  <img src="imagen.jpg" alt="Descripción de la imagen" />
  <div id="contenido">
    <p>Contenido dentro de un div.</p>
  </div>
</body>
</html>

HTML5

2014 - LA REVOLUCIÓN MODERNA

Características Principales

  • Nuevos elementos semánticos: header, footer, section, article, nav, aside, main
  • Soporte nativo para audio y video sin necesidad de plugins externos
  • Canvas para gráficos vectoriales, animaciones y juegos en 2D
  • APIs para aplicaciones web: geolocalización, almacenamiento local, drag and drop
  • Formularios mejorados con nuevos tipos de input (email, url, date, color, etc.)
  • Mejor soporte para dispositivos móviles con viewport y elementos responsivos
  • Sintaxis más flexible y forgiving que XHTML pero manteniendo buenas prácticas
  • Web Workers para ejecución de scripts en segundo plano
  • WebSockets para comunicación bidireccional en tiempo real
Ejemplo de código HTML5
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Página HTML5 Semántica</title>
</head>
<body>
  <header>
    <h1>Mi sitio web moderno</h1>
    <nav>
      <a href="#">Inicio</a>
      <a href="#">Acerca de</a>
    </nav>
  </header>
  <main>
    <article>
      <h2>Artículo principal</h2>
      <p>Contenido del artículo...</p>
      <video controls>
        <source src="video.mp4" type="video/mp4">
      </video>
    </article>
  </main>
  <footer>
    <p>Derechos reservados © 2023</p>
  </footer>
</body>
</html>
